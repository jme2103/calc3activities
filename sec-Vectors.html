    Section 1.2 Vectors

    Definition 1.2.1.

    A vector is a quantity that possesses the attributes of magnitude and direction.
    If a physical quantity has both magnitude and direction, it is called vectorial. If it has only magnitude, it is called scalar.

    Activity 1.2.1.

    Use your physical intuition to decide which of the following quantites are vectorial and which are scalar.
    1. temperature
    2. pressure
    3. velocity
    4. displacement
    5. length
    6. entropy
    7. density
    8. acceleration
    9. mass
    Graphically, a vector is represented as a line segment with an arrowhead placed on one end. The magnitude corresponds to its length and the direction corresponds to the direction that it points. Algebraically, a vector is represented by an \(n\)-tuple of numbers, called its components. For example, \({\bf v} = \langle v_1, v_2, \ldots, v_n \rangle\) is a vector in \(n\)-dimensional space. (N.B. Angled brackets denote a vector, whereas round brackets denote a point.)
    Graphically, the head of a vector is the end with the arrowhead, and the tail is the other end. When drawing a vector in Euclidean space, the point at which we place the tail is called the initial point, and the point at which we place the head is called the terminal point. If \(P\) is the initial point and \(Q\) is the terminal point, we often write \({\bf v} = \overrightarrow{PQ}\text{.}\)

    Activity 1.2.2.

    Suppose that the vector \({\bf v} = \langle -1, 0, 5 \rangle\) is represented as an arrow with terminal point \(Q = (4,-3,0)\text{.}\) What is the initial point \(P\) of the arrow?
    1. \(\displaystyle P = (5, 3, 5)\)
    2. \(\displaystyle P = (5, -3, -5)\)
    3. \(\displaystyle P = (-4, 0, 0)\)
    4. \(\displaystyle P = (3, -3, 5)\)
    The position vector of a point \(P\) is the vector whose initial point is the origin and whose terminal point is \(P\text{.}\)
    Two vectors are equal if they have the same magnitude and direction. Algebraically, this means the two vectors have the same components. Notice that neither the initial nor terminal point is mentioned in the definition of equality. So, graphically, this means the two vectors are equal if they can be superimposed by a translation.
    There are two operations that can be performed on vectors: addition and scalar multiplication. They are defined algebraically as:
    \begin{equation*} {\bf v} + {\bf w} = \langle v_1, v_2, v_3 \rangle + \langle w_1, w_2, w_3 \rangle = \langle v_1 + w_1, v_2 + w_2, v_3 + w_3 \rangle \end{equation*}
    \begin{equation*} c{\bf v} = c \langle v_1, v_2, v_3 \rangle = \langle cv_1, cv_2, cv_3 \rangle \end{equation*}

    Properties of vector operations.

    Let \(\vv\text{,}\) \(\vu\text{,}\) and \(\vw\) be vectors in \(\R^n\) and let \(a\) and \(b\) be scalars. Then
    1. \(\displaystyle \vv + \vu = \vu + \vv\)
    2. \(\displaystyle (\vv + \vu) + \vw = \vv + (\vu + \vw)\)
    3. The vector \(\vzero = \langle 0, 0, \ldots, 0 \rangle\) has the property that \(\vv + \vzero = \vv\text{.}\) The vector \(\vzero\) is called the zero vector.
    4. \((-1)\vv + \vv = \vzero\text{.}\) The vector \((-1)\vv = -\vv\) is called the additive inverse of the vector \(\vv\text{.}\)
    5. \(\displaystyle (a+b) \vv = a\vv + b\vv\)
    6. \(\displaystyle a(\vv + \vu) = a\vv + a\vu\)
    7. \(\displaystyle (ab) \vv = a(b\vv)\)
    8. \(1 \vv = \vv\text{.}\)

    Activity 1.2.3.

    Suppose that \(\vu = \langle 2, 1 \rangle\) and \(\vv = \langle 0, 3 \rangle\text{.}\)

    (a)

    Sketch \(\vu\text{,}\) \(\vv\text{,}\) and \(\vu + \vv\) on the same set of axes with initial point at the origin.

    (b)

    Sketch \(-\vu\) and \(3\vu\) on the same set of axes with initial point at the origin.

    (c)

    For any vectors \(\vu\) and \(\vv\text{,}\) which of the following best describes the vector \(\vu + \vv\text{?}\)
    1. It’s the vector along the diagonal of the parallelogram determined by \(\vu\) and \(\vv\) with initial point at corner opposite the initial points of \(\vu\) and \(\vu\text{.}\)
    2. It’s the vector along the diagonal of the parallelogram determined by \(\vu\) and \(\vv\) with initial point at terminal point of \(\vu\text{.}\)
    3. It’s the vector along the diagonal of the parallelogram determined by \(\vu\) and \(\vv\) with initial point at terminal point of \(\vu\text{.}\)
    4. It’s the vector along the diagonal of the parallelogram determined by \(\vu\) and \(\vv\) with the same initial point as both \(\vu\) and \(\vv\text{.}\)

    (d)

    For any vectors \(\vu\) and \(\vv\text{,}\) which of the following best describes the vector \(\vu - \vv\text{?}\)
    1. It’s the vector along the diagonal of the parallelogram determined by \(\vu\) and \(\vv\) with initial point at corner opposite the initial points of \(\vu\) and \(\vu\text{.}\)
    2. It’s the vector along the diagonal of the parallelogram determined by \(\vu\) and \(\vv\) with initial point at terminal point of \(\vu\text{.}\)
    3. It’s the vector along the diagonal of the parallelogram determined by \(\vu\) and \(\vv\) with initial point at terminal point of \(\vu\text{.}\)
    4. It’s the vector along the diagonal of the parallelogram determined by \(\vu\) and \(\vv\) with the same initial point as both \(\vu\) and \(\vv\text{.}\)

    (e)

    For any vectors \(\vu\) and \(\vv\text{,}\) which of the is the best best geometric description of the terminal points of the vectors \(\vu + t\vv\text{?}\)
    1. It’s the parallelogram determined by \(\vu\) and \(\vv\text{.}\)
    2. It’s the circle with center at the terminal point of \(\vu\) and with radius the magnitude of \(\vv\text{.}\)
    3. It’s the line with in the direction of \(\vv\) that goes through the terminal point of \(\vu\text{.}\)
    4. It’s the plane spanned by \(\vu\) and \(\vv\text{.}\)
