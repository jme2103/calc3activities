    Section 2.2 Tangent Planes

    Activity 2.2.1.

    Let \(f(x) = 3 - x^2\text{.}\)

    (a)

    Which of the following is an equation of the tangent line to the graph of \(f\) at the point \((1,2)\text{?}\)
    1. \(\displaystyle y = 2x + 4\)
    2. \(\displaystyle y = -2x + 4\)
    3. \(\displaystyle y = -2x + 3\)
    4. \(\displaystyle y = -2x\)

    (b)

    Use the tangent line to approximate \(f(1.1)\text{.}\)
    This is an example of one of the core ideas in calculus: if you zoom in enough most functions look like a linear function. The two caveats that we must include are that (1) the function is not too crazy (this property is called differentiability) and (2) the behavior is only valid in a small neighborhood of the point in question (we say the approximation is only valid locally).
    The next activity will guide you to correct linearization for a function of two variables.

    Activity 2.2.2.

    Let \(f(x,y) = 6 - \frac{x^2}{2} - y^2\) and focus on the point \((x_0, y_0) = (2,1)\text{.}\)

    (a)

    Compute \(f(2, 1)\text{.}\)

    (b)

    Find parameterizations \(\vr_1(t)\) and \(\vr_2(t)\) for the traces at \(x = 2\) and \(y = 1\)

    (c)

    On two separate sets of axes, sketch the graphs of the two traces and the tangent vectors to \(\vr_1'\) and \(\vr_2't\) at the time when they pass through the point \((2,1,f(2,1))\text{.}\) One set of axes represents the plane \(x = 2\text{,}\) and so its horizontal axis should be labelled \(y\text{.}\) The other represents the plane \(y = 1\text{,}\) so its horizontal axis should be labelled \(x\text{.}\)

    (d)

    The tangent plane to the graph of \(f\) at the point \((2,1)\) should contain the two vectors you plotted. Find an equation for this plane.

    (e)

    Use the equation of the tangent plane to approximate \(f(2.1, 1.1)\text{.}\)
    We now repeat the above procedure for any function \(f\) that is differentiable at the point \((x_0, y_0)\text{.}\) Consider the two parameterized curves
    \begin{equation*} \vr_1(t) = \lrangle{x_0, t, f(x_0, t)} \qquad \vr_2(u) = \lrangle{u, y_0, f(u, y_0)}. \end{equation*}
    Both curves are contained in the graph of \(f\text{,}\) and, when \(t = y_0\) and \(u = x_0\text{,}\) they intersect at the point \((x_0, y_0, f(x_0, y_0))\text{.}\) Their derivatives are
    \begin{equation*} \vr_1'(t) = \lrangle{0, 1, f_y(x_0, t)} \qquad \vr_2(t) = \lrangle{1, 0, f_x(t, y_0)}. \end{equation*}
    Because we are holding one of the variables constant in each case, the last component is the appropriate partial derivative. Finally, both vectors \(\vr_1'(y_0)\) and \(\vr_2'(x_0)\) are tangent of the graph of \(f\) at the point \((x_0, y_0, f(x_0, y_0))\text{,}\) we can define the tangent plane to be the plane containing these two vectors. First, we take their cross product
    \begin{equation*} \vr_1'(y_0) \times \vr_2'(x_0) = \det \begin{bmatrix} \vi \amp \vj \amp \vk \\ 0 \amp 1 \amp f_y(x_0, y_0) \\ 1 \amp 0 \amp f_x(x_0, y_0) \end{bmatrix} = \lrangle{f_x(x_0, y_0), f_y(x_0, y_0), -1}. \end{equation*}
    Using this the normal vector and \((x_0, y_0, f(x_0, y_0))\) gives the formula below.

    The tangent plane.

    If \(f(x,y)\) has continuous first-order partial derivatives, then the equation of the plane tangent to the graph of \(f\) at the point \((x_0,y_0,f(x_0,y_0))\) is
    \begin{equation} z = f(x_0,y_0) + f_x(x_0,y_0)(x-x_0) + f_y(x_0,y_0)(y-y_0).\tag{2.2.1} \end{equation}

    Activity 2.2.3.

    Which of the following is an equation of the tangent plane to \(z = x^2 - y^3\) at \((5,4)\text{?}\)
    1. \(\displaystyle z = 10(x - 5) - 39 - 3y^2(y - 4)\)
    2. \(\displaystyle z = 2x(x - 5) - 39 - 48(y - 4)\)
    3. \(\displaystyle z = 10(x - 5) - 39 - 48(y - 4)\)
    4. \(\displaystyle z = 2x(x - 5) - 3y^2(y - 4)\)

    Activity 2.2.4.

    1. Find the equation of the tangent plane to \(f(x,y) = 2 + 4x - 3y\) at the point \((1,2)\text{.}\) Simplify as much as possible. Does the result surprise you? Explain.
    2. Find the equation of the tangent plane to \(f(x,y) = x^2y\) at the point \((1,2)\text{.}\)

    Activity 2.2.5.

    An unevenly heated metal plate has temperature \(T(x,y)\) in degrees Celsius at the point \((x,y)\text{.}\) If \(T(2,1) = 110\text{,}\) \(T_x(2,1) = 12\text{,}\) and \(T_y(2,1) = -14\text{,}\) estimate the temperature at the point \((2.04, 0.95)\text{.}\)
    For a function of \(n\) variables, the tangent plane is an \(n\)-dimensional (affine, linear) subspace \(\R^{n+1}\text{.}\) When \(n=1\text{,}\) it’s a line in \(\R^2\text{;}\) when \(n=2\text{,}\) it’s a plane in \(\R^3\text{.}\) In general, we say that it’s a hyperplane in \(\R^{n+1}\text{.}\) The equation of the tangent hyperplane to a function \(w = f(x,y,z)\) at the point \((x_0,y_0,z_0)\) is
    \begin{equation*} w = f(x_0,y_0,z_0) + f_x(x_0,y_0,z_0)(x-x_0) + f_y(x_0,y_0,z_0)(y-y_0) + f_z(x_0,y_0,z_0)(z-z_0). \end{equation*}

    Activity 2.2.6.

    Find the tangent hyperplane to \(f(x,y,z) = zx^2 + y^2\) at the point \((1,2,1)\text{.}\)
